# HedgeIQ Frontend

Frontend application for HedgeIQ, a real-time options greeks analytics platform for cryptocurrency derivatives.

## Overview

HedgeIQ provides live options greeks analysis and gamma exposure (GEX) visualization for crypto markets. The frontend displays real-time dashboards for dealer positioning, strike-level greeks, and GEX profiles sourced from the HedgeIQ backend API.

## Key Features

- Real-time GEX profile visualization
- Options flow dashboard with live updates
- Strike-level greeks heatmaps
- Dealer positioning charts
- Responsive design for desktop and mobile

## Tech Stack

- HTML / CSS / JavaScript
- Chart.js for data visualization
- REST API integration with HedgeIQ backend

## Getting Started

1. Clone the repository
2. Ensure the HedgeIQ backend is running
3. Open `index.html` in your browser or deploy to any static hosting provider

## Deployment

Can be deployed to Netlify, Vercel, or any static file hosting service.
